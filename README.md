# Проект №2. Примитивная база данных

## Обзор
Этот проект представляет собой полноценое консольное приложение, имитирующее работу с базой данных.

Приложение написано на Python с применением некоторых внешних библиотек (в учебных целях).

## Рекомендации по установке и запуску
Приложение функционально на операционных системах GNU/Linux с версией ядра не ниже 6.8. Также будет не лишним владеть базовыми навыками работы с терминалом в Linux.

- Проверьте, что на вашей системе установлен интерпретатор Python с версией не ниже 3.12.10
  - Если версия вашего интерпретатора ниже, установите нужную версию в системе по инструкции в материале №3 раздела [Вспомогательные ссылки](#vspomogatelnye-ssylki). Не забудьте перейти на установленную версию!
- Скачайте архив .tar.gz и извлеките его содержимое в удобную вам папку
- В терминале откройте папку с файлами проекта
- Для установки приложения введите команду `make install`
- Для запуска приложения введите команду `make project`
